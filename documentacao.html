Visão Geral

O Sistema de Controle de Manutenção de Colheitadeiras é uma aplicação web desenvolvida em Python com o framework Flask, projetada para gerenciar todos os aspectos relacionados à manutenção de colheitadeiras agrícolas, com foco especial em equipamentos da marca John Deere.

O sistema permite o controle completo de manutenções preventivas e corretivas, monitoramento de horas de trabalho através do horímetro, registro de trocas de óleo e gestão de estoque de peças e produtos necessários para as manutenções.

Funcionalidades Principais

  • Cadastro de Colheitadeiras: Registro completo com modelo, ano, número de frota e status operacional.
  • Manutenções Preventivas: Agendamento baseado em horas de uso ou datas específicas.
  • Manutenções Corretivas: Registro de falhas, soluções aplicadas e peças substituídas.
  • Registros de Horímetro: Acompanhamento das horas de trabalho de cada equipamento.
  • Trocas de Óleo: Controle de trocas realizadas e programação das próximas trocas.
  • Gestão de Estoque: Controle de peças e produtos com alertas de níveis mínimos.
  • Relatórios: Geração de relatórios detalhados sobre manutenções, uso de peças e horas trabalhadas.
  • Controle de Usuários: Diferentes níveis de acesso para administradores, técnicos e operadores.

Estrutura do Sistema

O sistema foi desenvolvido seguindo o padrão MVC (Model-View-Controller) e utiliza as seguintes tecnologias:

  • Backend: Python com Flask
  • Banco de Dados: SQLite (desenvolvimento) / MySQL (produção)
  • ORM: SQLAlchemy
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Autenticação: Flask-Login

Gestão de Colheitadeiras

O módulo de gestão de colheitadeiras permite o cadastro completo dos equipamentos com as seguintes informações:

  • Modelo: Identificação do modelo da colheitadeira (ex: John Deere S700)
  • Ano: Ano de fabricação do equipamento
  • Número de Frota: Identificador único da colheitadeira na frota
  • Status: Estado atual do equipamento (Ativa, Em Manutenção, Inativa)

Através deste módulo, é possível:

  • Listar todas as colheitadeiras cadastradas
  • Adicionar novas colheitadeiras
  • Editar informações de colheitadeiras existentes
  • Visualizar detalhes completos de cada equipamento
  • Acessar o histórico de manutenções, trocas de óleo e registros de horímetro

Manutenções

Manutenções Preventivas

O sistema permite o agendamento e acompanhamento de manutenções preventivas baseadas em horas de uso ou datas específicas. Para cada manutenção preventiva, são registradas as seguintes informações:

  • Colheitadeira: Equipamento que receberá a manutenção
  • Descrição: Detalhes sobre o tipo de manutenção a ser realizada
  • Data Agendada: Data programada para a manutenção
  • Horímetro Agendado: Valor do horímetro para realizar a manutenção
  • Status: Estado atual da manutenção (Pendente, Realizada, Cancelada)
  • Data de Realização: Data em que a manutenção foi efetivamente realizada
  • Observações: Informações adicionais sobre a manutenção realizada

Manutenções Corretivas

O módulo de manutenções corretivas permite o registro de falhas e reparos realizados nos equipamentos. Para cada manutenção corretiva, são registradas as seguintes informações:

  • Colheitadeira: Equipamento que apresentou a falha
  • Descrição da Falha: Detalhes sobre o problema identificado
  • Data de Início: Data em que a manutenção corretiva foi iniciada
  • Status: Estado atual da manutenção (Em Andamento, Concluída)
  • Data de Conclusão: Data em que a manutenção foi concluída
  • Solução: Descrição da solução aplicada para resolver o problema
  • Peças Substituídas: Lista de peças utilizadas no reparo

Registros de Horímetro

O módulo de registros de horímetro permite o acompanhamento das horas de trabalho de cada colheitadeira, informação essencial para o planejamento das manutenções preventivas e trocas de óleo.

Para cada registro de horímetro, são armazenadas as seguintes informações:

  • Colheitadeira: Equipamento ao qual o registro se refere
  • Data do Registro: Data em que a leitura foi realizada
  • Valor do Horímetro: Número de horas registradas no equipamento

O sistema utiliza esses registros para:

  • Calcular a média de uso diário de cada equipamento
  • Prever quando serão atingidos os valores para manutenções programadas
  • Gerar alertas quando um equipamento está próximo de atingir o valor para manutenção
  • Fornecer dados para relatórios de utilização da frota

Trocas de Óleo

O módulo de trocas de óleo permite o registro e acompanhamento das trocas realizadas em cada colheitadeira. Para cada troca de óleo, são registradas as seguintes informações:

  • Colheitadeira: Equipamento que recebeu a troca de óleo
  • Data da Troca: Data em que a troca foi realizada
  • Horímetro: Valor do horímetro no momento da troca
  • Tipo de Óleo: Especificação do óleo utilizado (ex: 15W40)
  • Quantidade (litros): Volume de óleo utilizado
  • Próxima Troca (horímetro): Valor do horímetro para a próxima troca

O sistema utiliza essas informações para:

  • Gerar alertas quando um equipamento está próximo da próxima troca de óleo
  • Calcular o consumo médio de óleo por equipamento
  • Fornecer histórico completo de trocas para cada colheitadeira
  • Auxiliar no planejamento de compras de óleo para o estoque
